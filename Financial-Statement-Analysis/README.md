# Financial Statement Analysis – YETI Holdings, Inc.

## Overview

This project is a comprehensive financial and strategic analysis of YETI Holdings, Inc. completed as part of an advanced accounting and finance course.

The project evaluates YETI's historical financial performance (2018–2024), develops five-year financial projections, analyzes the company's competitive environment, and provides strategic recommendations supported by quantitative financial analysis.

## Project Highlights

- Analyzed seven years of financial statements (2018–2024)
- Calculated liquidity, solvency, profitability, and efficiency ratios
- Completed a full DuPont analysis
- Built five-year financial projections (2025–2029)
- Developed both Base Case and Improved Case forecasts
- Performed SWOT, PESTEL, and Porter's Five Forces analyses
- Proposed strategic recommendations supported by financial modeling
- Presented findings in both a written consulting report and executive presentation
