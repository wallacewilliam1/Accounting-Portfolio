# AAON Financial Modeling & Forecasting

## Overview

This project demonstrates the development of a three-year financial forecasting model for AAON, Inc. using audited SEC filings. The model recreates historical financial statements, develops forecast assumptions, projects future operating performance, and summarizes results through an executive dashboard.

---

## Project Objectives

- Reconstruct historical financial statements
- Build a dynamic forecasting model
- Forecast operating performance for 2026–2028
- Present financial results in an executive dashboard

---
